1. Navigate to FreeSQL.com and sign in or create a new account <freesql-button>

   ![click-sign-in-for-freesql](../../../building-blocks/tasks/freesql/images/click-sign-in-for-freesql.png " ")  

     ![click-create-new-account-button](../../../building-blocks/tasks/freesql/images/click-create-new-account-button.png " ")  

     ![creating-an-oracle-account-form](../../../building-blocks/tasks/freesql/images/creating-an-oracle-account-form.png " ")

     ![sign-in-with-existing-or-new-credentials](../../../building-blocks/tasks/freesql/images/sign-in-with-existing-or-new-credentials.png " ")  

     ![oracle-mfa-authorize-in-second-device](../../../building-blocks/tasks/freesql/images/oracle-mfa-authorize-in-second-device.png " ")

2. Once logged in, click the <strong>Connect with [rotating language option]</strong> button.

   ![click-connect-with-button](../../../building-blocks/tasks/freesql/images/click-connect-with-button.png " ")

3. Your new FreeSQL connection details will appear. Your password will appear only once. Copy the SQLcl connect string for future use. If you need a new password, click <strong>&circlearrowleft; Regenerate</strong>.

   ![schema-connection-details-for-freesql](../../../building-blocks/tasks/freesql/images/schema-connection-details-for-freesql.png " ")  
  
    ![regenerating-new-password](../../../building-blocks/tasks/freesql/images/regenerating-new-password.png " ")

<if type="vscode">

4. SQL Developer for VS Code Extension already installed? Take note of your password, click the <strong><span style="display:inline-block; transform: rotate(90deg);">&#8689;</span> Create a SQL Developer for VSCode connection</strong> button, and continue to the next task. Visual Studio Code should open automatically.

  ![create-a-sql-connection-in-vs-code-button](../../../building-blocks/tasks/freesql/images/create-a-sql-connection-in-vs-code-button.png " ")  

5. If you do not yet have the SQL Developer for VS Code Extension installed, note your FreeSQL credentials for later use:
    - Username
    - Password
    - Hostname
    - Port
    - Service Name

   ![sqlcl-connection-details-for-later-use](../../../building-blocks/tasks/freesql/images/sqlcl-connection-details-for-later-use.png " ")
   
</if>
